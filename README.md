# Steel Sheet Defect Classification
This project intended to classify the 8 types of defects in the Steel sheets named as 

 1.crazing
 
 2.folding 
 
 3.inclusion
 
 4.original 
 
 5.patch
 
 6.pitted surface
 
 7.rolled-in scale
 
 8.scratch.
 
Using the Deep Convolution Neural Network or Deep neural Network.
